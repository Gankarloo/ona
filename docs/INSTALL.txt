Here are some basic notes on installation of the system.

REQUIREMENTS:
  An apache server
  A mysql -server
  PHP
	php-mysql module is required
	php-gmp module is recomended

INSTALL:

1. Download the latest file from http://opennetadmin.com/downloads and place it in /opt
2. cd /opt
3. tar -zxvf opennetadmin-1.0.tar.gz


4. add a symlink in your web server root that points "ona -> /opt/ona/www"
   For example: ln -s /opt/ona/www /var/www/ona

   -or-

   Add this to your apache configuration file in the approrpiate location:

    Alias /ona "/opt/ona/www/"
    <Directory "/opt/ona/www/">
        Options Indexes MultiViews FollowSymLinks
        AllowOverride None
        Order allow,deny
        allow from all
    </Directory>

5. Point your broswer to http://<servername>/ona
